# ⚽ FutWear Store

E-commerce de camisas de futebol desenvolvido com arquitetura fullstack utilizando React no frontend e Node.js + Express no backend.

Projeto criado com foco em organização, escalabilidade e boas práticas de desenvolvimento.

---

## 🚀 Tecnologias Utilizadas

### Frontend
- React
- Vite
- CSS
- Fetch API

### Backend
- Node.js
- Express
- CORS
- Estrutura de rotas organizada

---

## 🧱 Estrutura do Projeto


futwear-store/
├── client/ # Frontend React
├── server/ # Backend Express


---

## ⚙️ Como Rodar o Projeto

### 1️⃣ Clonar o repositório


git clone https://github.com/SEU-USUARIO/futwear-store.git

cd futwear-store


---

### 2️⃣ Rodar o Backend


cd server
npm install
npm start


Servidor rodando em:


http://localhost:3000


---

### 3️⃣ Rodar o Frontend

Em outro terminal:


cd client
npm install
npm run dev


Aplicação rodando em:


http://localhost:5173
